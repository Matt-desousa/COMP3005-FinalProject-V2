# Matthew de Sousa - 101266991

# Details
This is a simple gym membership and class booking system.
It allows members to book classes, track health statistics, create exercise plans, and track their progress.
It allows trainers to view their clients, and set their availability.
It allows admins to manage classes, bookings, equipment, and payment and billing.

# Installation
1. Clone the repository
2. Install Java 17
3. Run the DDL.sql file in the /SQL folder in pgAdmin 4 to create the databases
4. Run the DML.sql file in the /SQL folder in pgAdmin 4 to populate the databases
5. Run the Main.java file and have fun!
